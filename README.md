# Data Fast Food - Pasrur

This is the official responsive website for **Data Fast Food**, located on Pasrur Road, Pasrur.  
It showcases the menu, special burgers, shawarma platters, fries, promotions, and contact information.  

## Features
- Menu display with popular items  
- Special offers and promotions  
- Location and contact details  
- Responsive design for desktop and mobile
